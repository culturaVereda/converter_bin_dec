# converter_bin_dec
pagina que utiliza html e css para realizar um conversor de números decimais para binários e vice versa


## O sistema numérico binário, desenvolvido pelo matemático e filósofo alemão Gottfried Wilhelm Leibniz no século XVII, utiliza apenas dois dígitos: 0 e 1. Nesse sistema, cada valor de posição representa uma potência de 2. A aplicação mais comum do binário está na tecnologia da computação, onde a linguagem de programação e a codificação digital se baseiam nesse sistema. Além disso, a simplicidade do binário torna-o ideal para representar informações em dispositivos eletrônicos, com 0 indicando “desligado” e 1 indicando “ligado”
